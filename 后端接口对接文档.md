# 后端接口对接文档

## 🔗 后端信息

### 基础配置
- **后端框架**: 项目 Spring Boot (Pharmacy-Vue-Pro)
- **后端地址**: `http://localhost:48080`
- **API 前缀**: `/admin-api`
- **完整地址**: `http://localhost:48080/admin-api`

### 响应格式
```typescript
{
  code: number      // 0: 成功, 其他: 失败
  data: any        // 响应数据
  msg: string      // 响应消息
}
```

### 分页响应
```typescript
{
  code: 0,
  data: {
    list: T[],     // 数据列表
    total: number  // 总条数
  },
  msg: "success"
}
```

## 📝 接口方法映射

### 项目框架标准接口

#### 分页查询
```java
@GetMapping("/page")
public CommonResult<PageResult<T>> getPage(PageReqVO reqVO)
```

#### 列表查询
```java
@GetMapping("/list")
public CommonResult<List<T>> getList(ListReqVO reqVO)
```

#### 详情查询
```java
@GetMapping("/get")
public CommonResult<T> get(@RequestParam("id") Long id)
```

#### 新增
```java
@PostMapping("/create")
public CommonResult<Long> create(@RequestBody CreateReqVO reqVO)
```

#### 修改
```java
@PutMapping("/update")
public CommonResult<Boolean> update(@RequestBody UpdateReqVO reqVO)
```

#### 删除
```java
@DeleteMapping("/delete")
public CommonResult<Boolean> delete(@RequestParam("id") Long id)
```

#### 导出
```java
@GetMapping("/export-excel")
public void exportExcel(ExportReqVO reqVO, HttpServletResponse response)
```

## 🔐 认证机制

### Token 格式
```
Authorization: Bearer ${token}
```

### 登录接口
```typescript
POST /system/auth/login
{
  username: string
  password: string
}

Response:
{
  code: 0,
  data: {
    accessToken: string,
    refreshToken: string,
    expiresTime: number,
    userId: number
  }
}
```

### 获取用户信息
```typescript
GET /system/auth/get-permission-info

Response:
{
  code: 0,
  data: {
    user: {
      id: number,
      username: string,
      nickname: string,
      avatar: string
    },
    roles: string[],
    permissions: string[]
  }
}
```

## 🚀 对接步骤

### 1. 更新环境变量
```env
# .env.development
VITE_APP_BASE_API=http://localhost:48080/admin-api
```

### 2. 更新 API 路径
需要批量更新所有 API 文件的路径，使其与后端一致。

### 3. 更新响应处理
后端使用 `CommonResult` 包装，需要在 `request.ts` 中处理。

### 4. 更新认证流程
- 登录接口: `/system/auth/login`
- 获取用户信息: `/system/auth/get-permission-info`
- Token 存储: `accessToken`
