## 📋 注意事项

### 1. 跨域配置
如果遇到跨域问题，需要在后端配置 CORS：
```java
@Configuration
public class CorsConfig {
    @Bean
    public CorsFilter corsFilter() {
        // 允许 http://localhost:5173
    }
}
```

### 2. Token 存储
- 登录成功后，Token 存储在 localStorage
- Key: `token`
- Value: `accessToken`

### 3. 权限验证
后端使用 `@PreAuthorize` 进行权限验证，前端需要：
- 登录后获取权限信息
- 根据权限显示/隐藏菜单和按钮

### 4. 文件上传
文件上传接口需要单独配置：
```typescript
POST /infra/file/upload
Content-Type: multipart/form-data
```
