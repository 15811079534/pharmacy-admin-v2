### 环境变量
```env
VITE_APP_BASE_API=http://localhost:48080/admin-api
VITE_APP_PORT=5173
```

### 请求头配置
```typescript
headers: {
  'tenant-id': '1',              // 租户ID（必需）
  'Authorization': 'Bearer ${token}',  // Token
  'Content-Type': 'application/json'
}
```

```typescript
{
  code: 0,           // 0: 成功
```

## ⚠️ 已知问题

### 1. 租户ID配置
**问题**: 后端需要租户ID才能访问
**解决**: 已在请求拦截器中自动添加 `tenant-id: 1`

### 2. 微信配置
**问题**: 微信公众号 appid 未配置导致启动失败
**解决**: 使用 `--spring.profiles.active=local` 禁用微信配置

### 3. 默认密码
**问题**: 初始密码可能不是 admin123
**解决**: 查看数据库或使用 SQL 脚本重置密码
